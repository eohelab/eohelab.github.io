# EOHE Lab website

Public website for the Earth Observation & Human Environment Lab (EOHE Lab),
School of Remote Sensing Science and Technology, Sun Yat-sen University.

- Website: <https://eohelab.github.io/>
- Chinese-mainland fallback: <https://sysu-eohe-2026-d9gc38h0203342f7d-1453818758.tcloudbaseapp.com/>
- Maintenance guide: [MAINTENANCE.md](MAINTENANCE.md)

The site is maintained as framework-independent HTML, CSS, and JavaScript. A
push to `main` runs the GitHub Pages workflow and publishes the portable static
build automatically.
